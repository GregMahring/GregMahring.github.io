MECHANICAL DESIGN

Handle Height Adjustor for a Walk Behind Lawn Mower

For this project, I was given the task of designing a user friendly system to adjust the height of a walk behind lawn mower handle that was effective. The system needed to have 3 height positions and be strong enough to withstand the rigors of mowing a lawn. On top of that, the system needed to fit with the design aesthetic that was established for the rest of the mower line. This system utilizes two pieces with a "tooth" system to index to the 3 desired positions. A toggle lever is used to disengage the teeth to allow for rotation and to engage the teeth and lock the system. It took several iterations to ensure a proper fit with the handle and the two pieces of the system. Collaboration with the Industrial Design department was also key to ensure the aesthetic aspects of the project were met.

Triloy Mower Rear Baffle and Lower Handles

The task for this project was to create a structural rear piece for a lawn mower that incorporated the rear discharge chute, housed the self- propelled transmission and acted as a means to attach the handle of the lawn mower. This project also involved designing the lower handle pieces which also housed half of the mechanism to adjust the handle height to 3 separate positions. This system was to be used on the top of the line lawn mower series sold for the European market, so the components needed to be strong and fit the aesthetic requirements of the mower. These components were made from die cast aluminum for strength and weight properties. Collaboration with the Industrial Design department was key to obtaining the aesthetics necessary for the project while also maintaining proper manufacturability.

Blade Engagement Lever and Housing for Electric Lawnmower

The concept behind this project was to develop a means for engaging the blade for an electric lawn mower. To conform to safety regulations for the intended market, the blade could not be engaged by a direct action, but needed a secondary action by the user. By depressing the button, it nests with the lever. Then as the lever is pulled, the button rotates and the finger on the button triggers the electric switch. This project also needed to fit within the aesthetic scope of the project, which involved a great deal of collaboration with the Industrial Design department.

Auxiliary Mirror for Heavy Truck

The task for this project was to develop an Auxiliary Mirror system for an on-highway heavy truck that was in development. The Auxiliary Mirror mounts to the front of the hood and allows the driver to see areas that are not visible with the Primary Mirror. There were many safety standards and strength requirements that needed to be met to make the project successful. The mirror needed to cover a specific field of view and also have adjustability. The arm and mounting features were held to very strict strength and vibration requirments, all under very tight time constraints.

Virtual Closet Web Application

The purpose of this project was to create a web application that allows the user to manage items of clothing within multiple categories. The user needed to have the ability to add, view, edit and remove items of clothing in their wardrobe, as well as view an image for each item created.
